========================
資料集層級之詮釋資料項目
========================

除了 CKAN 軟體套件預設提供的詮釋資料欄位外，CKAN 亦允許開發者可以視其平台所蒐集資料集之特性，擴充對應之詮釋資料項目。而本平台「研究資料公眾寄存所」依據所蒐集之研究資料集特性，規劃資料集層級之詮釋資料項目，計有「基本資訊」、「描述資訊」與「管理資訊」三大部分。
